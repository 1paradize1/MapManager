DESCRIPTION
------------------------------------------------------------
MapManager is a professional-grade toolkit designed to simplify texture management for choosen objects Autodesk 3ds Max. Unlike 
standard tools, MapManager provides a deep, hierarchical look into your scene's materials, allowing you to audit, 
fix, and optimize assets with surgical precision.

Whether you are preparing a scene for a render farm, cleaning up a merged model, or optimizing heavy textures, 
MapManager automates the routine and saves hours of manual work.


KEY FEATURES
------------------------------------------------------------
✔ DEEP SCAN: Detects textures in Standard, V-Ray, and Corona materials, including complex MultiMaps and Multi-Sub levels.

✔ SMART RESOLVER: Automatically locates missing files in project folders, subfolders, or custom library paths.

✔ PIPETTE TOOL: Scan textures only from selected polygons (Editable Poly level). Ideal for massive objects.

✔ INSTANCE DETECTION (·): Visual indicator for instanced maps. Know exactly which changes will affect multiple slots.

✔ PHYSICAL FILE CONTROL: Rename, Move, or Save textures directly from the UI with automatic path updating in scene.

✔ BUILT-IN OPTIMIZER: Convert any format to JPG or Resize high-res textures with a smart preview.

✔ PHOTOSHOP INTEGRATION: Open selected or all unique textures in Adobe Photoshop with one click.

✔ SMART SEARCH: Find and select objects in your scene by texture filename or by high resolution (e.g., all > 4K). Also missing textures search.

✔ ADVANCED UNDO (⤴): Full support for [Ctrl+Z], including the ability to reverse physical file renames on disk!


SUPPORTED VERSIONS
------------------------------------------------------------
Autodesk 3ds Max: 2016 – 2026
Render Engines: Standard, V-Ray, Corona.
Geometry: Editable Poly, Editable Mesh, Edit Poly (Pipette: Poly only).


INSTALLATION
------------------------------------------------------------
1. Drag and drop the "MapManager_v1.0.mzp" file into any 
   3ds Max viewport.
2. In the installer window, click "Install Plugin".
3. Restart 3ds Max (recommended).
4. Go to: Customize -> Customize User Interface -> Toolbars.
5. Select Category: "Paradize tools" and drag "MapManager" 
   to your toolbar.


UNINSTALL
------------------------------------------------------------
1. Drag and drop the same .mzp file into the viewport.
2. Click "Uninstall". All files and settings will be removed.
